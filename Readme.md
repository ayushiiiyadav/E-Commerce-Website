# Zivara - Luxury Diamond Jewellery E-Commerce

A modern, elegant, and fully functional **luxury jewellery e-commerce website** built with pure HTML, CSS, and JavaScript. Zivara specializes in diamond pendants and neck chains with a premium user experience.

## ✨ Features

- Beautiful animated hero section with auto-sliding carousel
- Real-time product search and category filtering
- Fully functional **Shopping Cart** with quantity controls
- **Wishlist** system with persistent storage
- User **Authentication** (Sign up / Login)
- User Profile with Order History
- Responsive design (Mobile-first)
- Premium loading animation with gem spinner
- Toast notifications
- Smooth animations and micro-interactions
- LocalStorage-based data persistence

## 🛠 Technologies Used

- **HTML5**
- **CSS3** (Custom variables, modern layout, animations)
- **Vanilla JavaScript** (No frameworks)
- Google Fonts (`Cormorant Garamond` + `Inter`)
